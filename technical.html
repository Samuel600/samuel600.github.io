What is Html?

Html is a code used to structure a web page and it content and it consist of a series of element that can be used or enclose or wrap different part of the content to make it appear a certain way or act a certain way. Html is mainly used for static pages. Html works best with CSS for styling and JS for Functionality.

1989, Tim Berner Lee invented the web with HTML as its publishing language. The language evolved from its version1.0 with limited tags and capabilities to now HTML 5 which has more capabilities.

Anatomy of an HTML

THE CONTENT: is the content of the element which is the case is just text.

ATTRIBUTES: contains extra info about the element that will not appear in the content. e.g class attribute is an identifying name usedbto target element with style info. The class attribute allows you to give the element a non-unique identifier that can be used.

THE ELEMENT: the opening tag, the closing tag,and content comprises of element

Properties of an attribute

An attribute should always have the following:
space between the element name or the previous attribute, if the element already has one or more already
The attribute name followed by an equal sign
The attribute value wrapped by opening and closing quotation
HTML TAG: it is used creating an element
Self closing tags: Tags that are not required to have specific closing tags. an example would be line break or image tag
